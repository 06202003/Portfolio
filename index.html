Sebelum mempelajari bootstrap, diharapkan untuk mempelajari terlebih dahulu struktur HTML dan CSS.
Bootstrap adalah framework atau rangka kerja HTML, CSS, JavaScript yang banyak dipakai untuk mendesain sebuah website dengan responsive. Bootstrap memberikan kemudahan kepada penguna dengan desain yang menarik.
Framework Bootstrap bersifat open source dan diciptakan pada tahun 2011 oleh Mark Otto dan Jacob Thornton dari Twitter. Itulah mengapa dahulu Bootstrap dinamakan Twitter Blueprint.

Frequenty Asked Question

Kegunaan Bootstrap pada pengambangan website adalah sebagai rangka kerja untuk membuat tampilan website menjadi responsive.
  • Menciptakan website mobile friendly
  • Memudahkan pengembang untuk resize gambar pada website
  • Menambahkan elemen-elemen pada website dengan mudah dan cepat
  • Membuat website lebih interaktif

Berikut beberapa kelebihan Bootstrap yang membuatnya sebagai salah satu framework paling populer di dunia :
  • Ramah untuk pemula
  • Grid system yang canggih
  • Kompabilitas dengan web browser yang baru
  • Bersifat open source
  • Kebebasan kostumisasi pada website
  • Rutin diperbaharui
  • Tersedia dokumentasi yang lengkap
  • Memiliki komunitas yang aktif dan lengkap

Berikut beberapa kelemahan Bootstrap :
  • Elemen banyak digunakan oleh pengembang website lain
  • Dapat memperlambat kinerja website
  • Memperlambat proses belajar
